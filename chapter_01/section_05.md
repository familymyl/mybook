# 变量与数据类型

***

## 1. 变量
字面量可以做多种不同的算术和逻辑运算，运算结果可以使用变量临时存储起来，变量在使用前需要定义其存储的数据类型。例如：

声明变量为int（整数）类型：
```
int a;
```

将2 + 3 的运算结果存入变量：
```
a = 2 + 3;
```
因为2和3都是整数，因此可以存入整数类型的变量中。

> 注意：java中可扩展的类型有无穷多个，但基本类型只有八种。
