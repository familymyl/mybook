# 运算符

---

## 1. 算数运算

java 支持基本的算术运算：加、减、乘、除、模（余数）

例如：

| 表达式 | 结果 | 说明 |
| :--- | :--- | :--- |
| 3 + 4; | 7 | 加法 |
| 3 - 4; | -1 | 减法 |
| 3 \* 4; | 12 | 乘法 |
| 12 / 4; | 3 | 整数除法 |
| 3 / 4; | 0 | 整数除法，整数除法会忽略所有小数部分 |
| 3 / 0; | ArithmeticException | 整数除法不能除以0，会出现异常 |
| 3.0 / 4.0 | 0.75 | 小数除法 |
| 3.0 / 0.0 | Infinity | 小数除法可以除以0.0 结果是正无穷大 |
| 10 % 3 | 1 | 求模(余数) |
