# 注释

***
## 1. 注释
所谓注释，不影响程序运行，只起到提示程序员的作用。换个角度说，注释只存在与 \*.java 文件中，在javac编译之后的 \*.class 文件中，注释已经消失了。

> 注意：注释可以加在代码的任意位置

## 2. 单行注释
例如：

```
// 这是单行注释1
public class HelloWorld {
    // 这是单行注释2
    public static void main(String[] args) {
        // 这是单行注释3
        System.out.println("你好，世界！"); 
    }

}
```
