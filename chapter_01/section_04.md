# 基本数据类型

***

#1. 字面量(literal)
所谓字面量就是用来在程序中表示不同类型值的表示方式

例如：

| 类型 | 值 | 说明 | JDK |
|---|---|---|---|
| 整数 | +10 | 正整数10 |   |
| 整数 | 10 | 正整数10(省略+号) |   |
| 整数 | -10 | 负整数-10 |   |
| 整数 | 010 | 八进制整数8 |   |
| 整数 | 0x10 | 十六进制整数16 |   |
| 整数 | 0b10 | 二进制整数2 | 7.0 |
| 长整数 | 10L | 正整数10 |  |
| 浮点数 | 1.0D | 双精度浮点数 |   |
| 浮点数 | 1.0 | 双精度浮点数(省略D) |   |
| 浮点数 | 1.0F | 单精度浮点数 |   |
| 浮点数 | 1.0E-3 | 单精度浮点数0.001 |   |
| 浮点数 | 1.0E+3 | 单精度浮点数1000.0 |   |
| 字符 | 'a' | 字符a(整数97) |   |
| 字符 | '0' | 字符0(整数48) |   |
| 字符 | '一' | 字符0(整数0x4e00) |   |
